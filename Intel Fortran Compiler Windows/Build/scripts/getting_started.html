Getting Started using cfast Build Scripts

Initial Setup

  1. Defining environment variables used by the scripts
    1. Open the Build\scripts directory and double-click on the file COPYconfig2home.bat. This copies the configuration file cfast_env.bat to your home directory.
    2. Click here to update the cfast_env.bat configuration file. (Define your cfast repository location.)

  2. Building software
    1. Install Intel Fortran.

  3. Connecting to other Computers
    • Install putty.
    • Set up ssh keys. (optional - to avoid typing passwords)
      1. run puttygen
      2. click on the SSH-2 RSA radio button
      3. click on the Generate button. Pick a pass-phrase.
      4. Save the private key on your local computer.
      5. Save the public key and copy it to the Linux cluster. Put the key at the end of the file authorized_key2 which is located in the .ssh directory in your home directory.
    • Add putty location to system PATH variable

  4. Creating Bundles
    1. Install Winzip
    2. Install Winzip command line tool
    3. Install Winzip self extractor.
    4. Add Winzip and Winzip self extractor locations to system PATH variable
